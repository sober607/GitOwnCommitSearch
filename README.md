# GitOwnCommitSearch

GitOwnCommitSearch is a .NET application that searches for specific commits in multiple Git repositories. It allows you to search for commits based on commit messages and diffs.

## Features

- Search for commits in multiple repositories.
- Filter commits by branch name.
- Search for specific terms in commit messages.
- Search for specific terms in commit diffs.

## Prerequisites

- .NET Framework 4.8
- LibGit2Sharp
